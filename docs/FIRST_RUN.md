# First Run

1. Install dependencies from `requirements.txt`.
2. Start the app with `python -m app.main`.
3. The app creates the local SQLite database automatically.
4. Sample devices are seeded with documentation-safe example addresses.
5. Open the Devices page to add your first IP-based monitor target.
6. Start monitoring from the dashboard toolbar.

## Notes

- Runtime database files are ignored by git.
- No agent is required on monitored endpoints.
- Discovery scans should target your own networks only.
