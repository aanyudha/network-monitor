# Roadmap

## Near term

- SNMP implementation for interfaces and traffic metrics
- Discovery fingerprinting improvements with richer vendor and banner detection
- Device groups and monitoring templates
- Better uptime charts and historical dashboards
- Export filters and scheduled report jobs
- Traffic visibility adapters for DNS, firewall, and proxy pipelines

## Future

- Packaging for Windows and Linux
- Notification integrations beyond desktop popups
- Topology mapping enhancements
- Optional distributed probe or agent mode
- Flow-based visibility integrations for NetFlow, sFlow, and IPFIX
